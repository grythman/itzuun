# ITZuun MVP Roadmap (30–60–90)

## Day 0–30: Foundation
**Goal:** Core marketplace loop ажиллуулах

### Deliverables
- OTP auth + JWT
- Role model (client/freelancer/admin)
- Profile CRUD
- Project CRUD + filtering
- Proposal submit/list/select
- Minimal admin endpoints

### Exit Criteria
- Client project post хийж чадна
- Freelancer proposal илгээж чадна
- Client freelancer сонгож `in_progress` төлөвт оруулна

## Day 31–60: Trust + Transactions
**Goal:** End-to-end secured transaction flow

### Deliverables
- Escrow deposit + admin approve
- Submit result + confirm completion
- Ledger entries (deposit/fee/release/refund)
- Chat + file upload
- Review + rating summary
- Basic dispute raise/resolve flow

### Exit Criteria
- 1 төслийн эхлэлээс дуусгалт хүртэл бүх урсгал ажиллана
- Escrow state transitions алдаагүй
- Admin dispute action ажиллана

## Day 61–90: Launch Readiness
**Goal:** Production launch + early growth

### Deliverables
- Freelancer verification workflow hardening
- Admin ops dashboard polish
- Audit logs / observability baseline
- UX improvements on project/proposal pages
- Landing + onboarding content
- Pilot rollout (first 20 projects)

### Exit Criteria
- Real users with real escrow transactions
- Support playbook for disputes
- KPI tracking for activation and completion

## KPI Tracking (Weekly)
- New freelancer signups
- Verified freelancer count
- Projects posted
- Proposal-to-hire conversion
- Escrow funded count
- Completion rate
- Avg rating, dispute rate
